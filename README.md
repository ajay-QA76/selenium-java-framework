# Selenium Java Framework

This project is a basic automation testing framework built using Selenium WebDriver, Java, TestNG and Maven.

## Features
- Page Object Model
- TestNG Execution
- Maven Build Management
- Reusable Utilities
- Reporting Support

## Tools & Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven
- GitHub

## Test Scenarios
- Login Test
- Search Functionality Test
- Add to Cart Test
